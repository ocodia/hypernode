hypernode

hypernode is a local-first canvas for shaping connected ideas fast. It autosaves in your browser and opens or saves as a JSON file you can control.

Start at speed

New hypernodes open with a starter node already in Zen edit mode, so you can begin typing immediately.

Connect cleanly

Use anchors to sketch directional relationships, reconnect edges in place, and move through the structure without breaking flow.

Frame the field

Draw frames to gather nearby nodes into spatial groups. The canvas previews membership while you drag.

Drop into Zen

Focus a node when you want a quieter editing surface for title, notes, and image updates without leaving the hypernode.
