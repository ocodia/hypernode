hypernode

About hypernode

hypernode is a local-first graph editor for visual thinking. Data is saved in your browser and can be imported or exported as JSON.

How to use

  • Double-click empty canvas or use + to create a node.
  • Drag nodes to move them and drag empty canvas to pan.
  • Use node anchors or selected edge endpoints to create/reconnect edges.

Keyboard shortcuts

  • Del or Backspace: delete selected node/edge.
  • Ctrl/Cmd + Z: undo.
  • Ctrl/Cmd + Y or Ctrl/Cmd + Shift + Z: redo.
  • Ctrl/Cmd + Enter: save and close node editor.
  • Esc: exit edit mode, cancel connect, or clear selection.
